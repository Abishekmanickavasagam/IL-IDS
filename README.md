# 🚀 Overview

IL-IDS (Incremental Learning Intrusion Detection System) is an adaptive cybersecurity framework designed to detect **evolving network threats in real-time** using **Incremental Learning**.

Traditional IDS models struggle with **concept drift** and newly emerging cyberattacks. IL-IDS solves this problem by continuously learning from new attack patterns without forgetting previously learned knowledge.

---

# ✨ Key Features

✅ Incremental Learning for evolving threats  
✅ Real-time intrusion detection  
✅ Memory Replay mechanism  
✅ Elastic Weight Consolidation (EWC)  
✅ Drift Detection for attack adaptation  
✅ High attack detection accuracy

---

# 🧠 Methodology

The system combines:

- **Multi-Layer Perceptron (MLP)**
- **Incremental Learning**
- **Elastic Weight Consolidation (EWC)**
- **Memory Replay**
- **Drift Detection**

to create a continuously adaptive cybersecurity model.
